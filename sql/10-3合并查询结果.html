合并查询结果：将多个select查询结果合并在一起。
		使用关键字union和union all
	union：将所有的查询结果合并在一起，去除相同的记录。
	union all:只是简单的合并

	select语句之间用union|union all进行连接。
	语法:select 语句1 union|union all 语句2.。。。
	
	union['ju:niən]:同盟。公会 联合 团结

select d_id from department union select d_id from employee;
select d_id from department union all select d_id from employee;

表名和表字段取别名：
	格式: 表名 表的别名
	格式：属性名 [as] 别名   （as可有可无）

select * from department dpart where dpart.d_id=1001;//表别名为dpart
select d_id as did,d_name as dname from department;//字段别名。。显示出来的表上的字段也是别名
select d_id as did,d_name as dname from department dpart where dpart.d_id=1001;//表别名和字段别名

使用正则查询：
	语法：属性名 regexp '匹配方式';
	匹配方式:
		^:字符串开始
		$:字符串结束
		.:代表字符串中任意一个字符，包括回车和换行
		[字符集合]:匹配"字符集合"中的任何一个字符
		[^字符集合]:除了"字符集合"中的任何一个字符
		s1|s2|s3:匹配s1 s2 s3中任何一个字符
		*:代表任何个字符
		+:至少1个字符
		{n}：字符串出现n次
		{m,n}:字符串出现至少m次，最多n次。
		

select * from info where name regexp "^Lamb";//查找字符以‘Lamb’开头的记录
select * from info where name regexp "amn$";//以字符amn结尾的记录
select * from info where name regexp "^A..y$";//以A开头，以y结尾，中间是2个任意字符的记录。
select * from info where name regexp "[ceo]";//name字段中包含c e o 3个字符中任何一个的记录。
select * from info where name regexp "[0-9a-c]";//name 中 数字和字母a b c 。
select * from info where name regexp "[^0-9]";//name 不能包含数字
select * from info where name regexp "must";name中包含must字符串的记录。
select * from info where name regexp "m*st";//*号位置 >=0个字符
select * from info where name regexp "m+st";//+号位置 至少1个字符 >=1个字符

select * from info where name regexp "a{3}";//a至少出现3次
select * from info where name regexp "a{1,3}";//a至少出现1次，最多3此
